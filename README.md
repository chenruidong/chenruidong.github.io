# Ruidong Chen Website

Static personal site prepared for GitHub Pages.

## Files

- `index.html`: main page
- `styles.css`: site styles
- `site-copy-review.md`: editable copy deck for proofreading
- `site-copy-review-zh.md`: editable Chinese copy deck for proofreading

## Local preview

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173`.

## GitHub Pages

Recommended GitHub repository name for a user site:

`chenruidong.github.io`

If you use a custom domain later, add a `CNAME` file when we wire deployment.
